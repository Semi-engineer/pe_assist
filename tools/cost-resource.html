คำอธิบายตัวแปร

  • Labor Cost / hr: ค่าแรงต่อชั่วโมง
  • Machine Cost / hr: ค่าใช้จ่ายเครื่องจักรต่อชั่วโมง
  • Overhead Cost / hr: ค่าใช้จ่ายแฝงต่อชั่วโมง
  • Cycle Time (sec): เวลาผลิตต่อชิ้น
  • Available Time (min): เวลาทำงานต่อกะ
  • Customer Demand (pcs): จำนวนสินค้าที่ลูกค้าต้องการ
  • Cost per Unit: ต้นทุนต่อหน่วยสินค้า
  • Line Capacity (pcs): กำลังการผลิตของสายการผลิต
  • Resource Utilization (%): สัดส่วนการใช้ทรัพยากรที่มีอยู่
