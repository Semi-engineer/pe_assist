คำอธิบายตัวแปร

  • Value Added Time (VA, min): เวลาที่เพิ่มคุณค่าให้กับผลิตภัณฑ์หรือบริการ
  • Non Value Added Time (NVA, min): เวลาที่ไม่เพิ่มคุณค่าและควรลดหรือกำจัดออก
  • VA %: สัดส่วนของเวลาที่เพิ่มคุณค่าเมื่อเทียบกับเวลารวม
  • NVA %: สัดส่วนของเวลาที่ไม่เพิ่มคุณค่าเมื่อเทียบกับเวลารวม
  • Initial Setup Time (min): เวลาที่ใช้ในการตั้งค่าเครื่องจักรหรือกระบวนการก่อนเริ่มผลิต
  • Reduced Setup Time (min): เวลาที่ใช้ในการตั้งค่าหลังจากมีการปรับปรุงกระบวนการตามหลัก SMED
  • Time Saved (min): เวลาที่ประหยัดได้จากการลดเวลาในการตั้งค่า
  • % Improvement: เปอร์เซ็นต์ของการปรับปรุงเวลาการตั้งค่าหลังจากใช้ SMED
