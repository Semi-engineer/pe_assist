คำอธิบายตัวแปร

  • USL: ขอบบน specification
  • LSL: ขอบล่าง specification
  • Process Mean (μ): ค่าเฉลี่ยของกระบวนการ
  • Process Std Dev (σ): ส่วนเบี่ยงเบนมาตรฐานของกระบวนการ
  • Units Produced: จำนวนสินค้าที่ผลิต
  • Number of Defects: จำนวนของเสีย
  • Opportunities per Unit: จำนวนโอกาสเกิด defect ต่อหน่วย
  • Cp: Process Capability Index - ความสามารถของกระบวนการในการผลิตให้อยู่ในขอบเขตที่กำหนด
  • Cpk: Process Capability Index - ความสามารถของกระบวนการที่คำนึงถึงความเบี่ยงเบนของค่าเฉลี่ย
  • PPM / DPMO: Parts Per Million / Defects Per Million Opportunities - จำนวนของเสียต่อหนึ่งล้านหน่วยหรือโอกาส
  • Six Sigma Level: ระดับคุณภาพที่แสดงถึงจำนวนของเสียที่คาดว่าจะเกิดขึ้นในกระบวนการ
